# Environmental Data Logging and Analysis System

## Objective
A Python OOP simulation that captures and analyzes environmental sensor data
with trend detection and alert generation.

## Features
- Sensor registration
- Measurement logging
- Daily averages
- Min / Max / Mean calculation
- Threshold-based alerts

## Technologies
- Python 3
- Object-Oriented Programming
- GitHub

## How to Run
```bash
cd src
python main.py
